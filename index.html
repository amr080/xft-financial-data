Introduction

Welcome to financial datasets, a developer-friendly stock market API.

​
Overview

We have data for 15,000+ public companies, going back 30+ years.

You can use our data to power your AI financial agents, portfolio management tools, stock analysis platforms, quantitative trading algorithms, and more.

You are welcome to save any data that you receive from our API. Our platform is built by developers for developers.

​
Getting Started

You can create an account and get an API key at financialdatasets.ai.

For testing purposes, data for the following tickers is free: AAPL, GOOGL, MSFT, NVDA, TSLA.
